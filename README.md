# vue-ToDoMVC
VUE.js的官方入门项目
# 可实现功能：  
+ 第一次打开就自动鼠标焦点在输入框  
+ 在输入框输入字符串，下面自动新增记录且里面有刚刚新增的字符串  
+ 双击某一项，可进入编辑模式，且自动焦点在此条数据  
  + ps：修改后  
    + 2.1 按esc键会退出编辑状态并且不会保存    
    + 2.2 按enter或将鼠标点击空白处，会自动保存
+ 单击某条记录前空白处，会显示已选中，可单击后面❌号删除此条记录  
+ 左下角会计算出未被选中的记录  
+ 单击下面3个按钮可实现上方显示出对应条件的记录  
+ 右下角可将选中数据批量删除
